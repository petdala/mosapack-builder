# Cricut First Hello Mini Validation v1

Date: 2026-07-09
Status: optional local validation plan; not a public product path.

## Purpose

Define a small local craft-cutter validation using the synthetic First Hello sample. This is for internal feasibility only and does not create a DIY template, download product, or public cutting-machine workflow.

## Scope

Validate:

- whether `0.5in` rounded-square cells can be cut cleanly at small scale
- whether printed color cells remain identifiable
- peelability and placement speed
- whether guide/map instructions are clear enough for a small build

Do not validate:

- public DIY template sales
- cutting-machine compatibility promises
- checkout
- pricing
- shipping
- production readiness

## Materials

- synthetic First Hello sample artwork
- test sticker paper
- craft cutter
- cutting mat
- printed placement guide or grid
- timing sheet

## Procedure

1. Print a small First Hello test sheet at actual size.
2. Cut `0.5in` rounded-square cells.
3. Weed and peel a small sample group.
4. Place 50-100 cells on a printed guide.
5. Record cut quality, peelability, placement errors, time per sticker, and color separation.
6. Stop if cells tear, curl, lose adhesive, or become too difficult to place.

## Pass / Fail

| Check | Pass | Fail |
| --- | --- | --- |
| cut quality | clean rounded-square cells | tearing, miscuts, lifted corners |
| peelability | cells peel by hand | cells tear or stick to liner |
| placement | cells can be placed accurately | repeated drift or frustration |
| color separation | colors remain distinguishable | color groups become confusing |
| time | measured and tolerable | too slow for target experience |

## Reporting

Record:

- cutter model
- material
- blade/depth settings
- print settings
- cut settings
- photos of sheet and placed sample
- timing and error notes

Do not publish customer-facing claims from this test alone.
