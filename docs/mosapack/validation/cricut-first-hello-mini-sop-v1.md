# Cricut First Hello Mini SOP v1

Date: 2026-07-09
Status: validation SOP; not public production.

## Purpose

Validate whether a Cricut Maker 3 workflow can support a small First Hello Mini learning batch. This is not a public production path.

## Sample Scope

- First Hello Mini
- 12 inch black-base board
- 24x24 grid
- 0.5 inch cells
- black board replaces black stickers
- Sono-4 Essential first
- optional Sono-6 Heirloom second

## 1. Cut Calibration

1. Confirm blade, mat, and material.
2. Run a small 10-20 cell cut with 0.5 inch rounded-square geometry.
3. Record material, setting, pressure, passes, mat condition, and blade.
4. Inspect corners, liner cut-through, lift, tearing, and curl.
5. Adjust only one variable at a time.

## 2. Material Test

Test each Sono-4 Essential color:

- Snow White
- Light Gray
- Mid Gray
- Shadow Gray

For each color:

1. Cut at least 25 cells.
2. Peel 10 cells by hand.
3. Weed 10 adjacent cells.
4. Score peel and weed from 1-5.
5. Photograph the sheet, weeded liner, and removed cells.

## 3. 100-150 Sticker Timed Build

1. Select a controlled First Hello Mini section.
2. Cut 100-150 cells across all four colors.
3. Start timer before peeling.
4. Place cells on black-base board.
5. Record time to cut, weed, and place.
6. Calculate seconds per sticker.
7. Record placement errors and frustration points.

## 4. Gray Separation Check

1. Place the four gray colors under normal room light.
2. Photograph from straight-on and at an angle.
3. Score gray separation from 1-5.
4. If colors collapse visually, test Sono-6 Heirloom as a second pass.

## 5. Photo Documentation Checklist

Capture:

- raw material sheet before cutting
- cut sheet before weeding
- weeded sheet
- peeled cells
- black-base board before placement
- partial placement at 50 cells
- finished timed section
- close-up of edges/corners
- lighting condition notes

## 6. Pass / Fail Thresholds

Pass candidate:

- peel score >= 4
- weed score >= 4
- gray separation score >= 4
- no repeated tearing or curling
- registration drift is measurable and acceptable for a small test
- time per sticker is recorded and not obviously impractical

Fail or downgrade:

- repeated tearing
- cells lift during weeding
- liner cuts through unpredictably
- gray groups cannot be distinguished
- placement time is clearly too slow for any beta batch

## Rule

One physical sample can downgrade or flag a SKU, material, or process. It cannot kill 24 inch Signature or future product sizes by itself.
