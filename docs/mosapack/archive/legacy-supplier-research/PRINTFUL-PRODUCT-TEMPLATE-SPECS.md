**Product Code**: PCSSKCS
**Size**: 8.5" × 11"

---

### **Template Specifications:**

**Print File Dimensions:**
- Width: 8.5 inches (2550 pixels @ 300 DPI)
- Height: 11 inches (3300 pixels @ 300 DPI)
- **Total canvas**: 2550 × 3300 pixels

**File Format:**
- Preferred: PNG (transparent background support)
- Alternative: JPG (if no transparency needed)
- Resolution: 300 DPI (REQUIRED for quality)
- Color Mode: RGB (NOT CMYK - Printful converts)

**Bleed:**
- No bleed required for kiss-cut stickers
- Design should stay within safe area (see below)

**Safe Area:**
- Keep all important content 0.125" (37.5 pixels @ 300 DPI) from edges
- This prevents important details from being cut off

---

### **Design Layout:**

**Your Mosaic Design:**
```

│  (0.125" margin)                    │
│   ┌─────────────────────────┐       │

│   │   (32×32 tiles example) │       │
│   │                         │       │
│   │                         │       │
│   │                         │       │
│   │   Each tile is kiss-cut │       │
│   │   (customer peels each) │       │
│   │                         │       │
│   └─────────────────────────┘       │
│  (0.125" margin)                    │
└─────────────────────────────────────┘
     8.5" × 11" total canvas
```

**Tile Specifications:**
- **32×32 grid**: Each tile is ~0.25" square
- **48×48 grid**: Each tile is ~0.17" square
- **64×64 grid**: Each tile is ~0.13" square

**Minimum tile size**: 0.1" (Printful kiss-cut minimum)

---

### **Color Requirements:**

**Color Profile**: sRGB IEC61966-2.1
**Bit Depth**: 8-bit per channel minimum
**Black**: RGB(0, 0, 0) - not pure black in CMYK
**White**: RGB(255, 255, 255)

**For best color matching:**
- Use sRGB color space (most common for web/photos)
- Don't use CMYK (Printful converts from RGB)
- Keep colors within sRGB gamut for accurate printing

---

- [ ] File is 2550 × 3300 pixels

- [ ] File format is PNG or high-quality JPG (90%+ quality)
- [ ] Color mode is RGB

- [ ] All tiles are at least 0.1" × 0.1" (minimum for kiss-cut)
- [ ] File size is under 100MB

## 🎨 **DESIGN BEST PRACTICES:**

### **For ALL Products:**

**Color Accuracy:**
- Use sRGB color profile (embedded in file)
- Calibrate your monitor if possible
- Colors on screen may look different than printed (normal!)
- Order test samples to see actual print colors

**Resolution:**
- ALWAYS use 300 DPI for print quality
- Never upscale low-res images (results in blurry prints)
- If source photo is low-res, use smaller grid (fewer tiles)

**File Size:**
- Keep files under 100MB (Printful limit)
- PNG files can be large - use JPG if over 100MB (90% quality minimum)

**Borders:**
- For posters/framed: No border needed (full-bleed looks best)
- For stickers: Optional border around tiles (helps with alignment)

---

### **Mosaic-Specific Tips:**

**Grid Size Selection:**
- **Sticker Sheet (8.5×11")**: 32×32 or 48×48 tiles max
  - Larger grids = smaller tiles = harder to peel
  - Keep tiles at least 0.1" × 0.1"

- **Posters (18×24")**: 48×48 or 64×64 tiles
  - Larger grids = more detail
  - Smaller grids = bolder, more abstract

- **Posters (24×36")**: 64×64 or 96×96 tiles
  - Maximize detail with larger canvas

**Tile Spacing:**
- Add 1-2 pixel gap between tiles for easier kiss-cutting (stickers)
- No gaps needed for posters (looks cleaner)

**Numbering (Stickers Only):**
- Add small numbers to tiles (optional, helps with assembly)
- Use 4pt font minimum
- Place in corner of each tile
- Light gray color (visible but not distracting)
