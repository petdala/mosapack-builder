Phase-by-Phase Validation Plan

Phase 1: Sticker Validation (Week 1-2) • Budget: $200

Actions:
  • Order 50 sample stickers from StickerYou (5 designs, 10 each)
  • Set up basic Shopify store with Printful POD integration
  • Run $100 Meta ads targeting home décor buyers
  • Collect email signups, measure CTR and CAC
Success Metrics: CAC < $20 • CTR > 2% • 20+ email signups

Phase 2: Brick Prototype (Week 2-4) • Budget: $400

Actions:
  • Order 100 pieces × 20 colors from MyGobricks ($300)
  • Purchase Nix Mini 3 for color QC ($100)
  • Build 2-3 full prototypes (32×32, 48×48)
  • Test assembly time, instructions clarity
  • Measure brick colors against palette library
Success Metrics: Assembly < 2hrs for 48×48 • Color ΔE < 3.0 • Clutch power test pass

Phase 3: Packaging & Framing (Week 3-4) • Budget: $200

Actions:
  • Test 3 frame options: budget ($12), mid ($22), premium ($44)
  • Design packing strategy for brick kits (color-coded bags)
  • Create instruction PDF with Builder PRO v3
  • Photograph completed mosaics in frames
  • Test connector systems for multi-plate mosaics
Success Metrics: Professional photos ready • Frame fits perfectly • Instructions clear to non-builders

Phase 4: Market Testing (Week 4-6) • Budget: $200

Actions:
  • Launch landing page with all three product options
  • Run split A/B test ads on Meta/Pinterest
  • Target: 100 website visitors per pathway
  • Collect pre-orders or measure conversion intent
  • Survey respondents on price sensitivity
Success Metrics: Conversion rate > 2% • Average order value > $40 • NPS survey > 60
