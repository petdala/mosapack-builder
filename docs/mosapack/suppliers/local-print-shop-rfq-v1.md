# Local Print Shop RFQ v1

Date: 2026-07-09
Status: supplier feasibility inquiry; not a production order.

## Purpose

Ask a local print shop whether it can produce short-run mixed sticker sheets and optionally full-kit components for MosaPack proof validation.

## Requested Sample

Starter validation target:

- `0.5in` rounded-square sticker cells
- `24x24` / `12in` proof target
- custom mixed sticker sheets per design
- synthetic sample artwork only

## Primary Questions

1. Can you print and kiss-cut `0.5in` rounded-square stickers on sheets?
2. Can you print custom mixed sheets per order from supplied PDF artwork?
3. What bleed do you require for edge-to-edge color on a `0.5in` rounded-square die?
4. What registration tolerance can you hold on short-run digital sticker sheets?
5. Can you test both `0.03in` and `0.05in` bleed samples?
6. What matte, weatherproof, removable, and permanent adhesive materials do you offer?
7. Can you label sheets by section, color, or sequence?
8. Can you package sticker sheets with a printed placement board and instruction card?

## Optional Full-Kit Questions

1. Can you print or source a `12in` placement board / printed grid board?
2. Can the board accept sticker adhesion without curling or peeling?
3. Can you package board, sticker sheets, and instruction card together?
4. Can you support `16in` boards or tiled `24in` panels later?

## Quote Request

Please quote sample, 10-unit, and 100-unit test quantities for:

- mixed sticker sheets only
- mixed sticker sheets plus instruction card
- mixed sticker sheets plus placement board plus instruction card

Do not include customer private images. This is a feasibility request only.
