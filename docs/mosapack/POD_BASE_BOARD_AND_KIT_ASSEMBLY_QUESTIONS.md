# POD Base Board and Kit Assembly Questions

Date: 2026-07-07
Status: supplier question addendum; production deploy not approved.

## Purpose

MosaPack is validating not just sticker sheets, but a complete sticker mosaic kit. The full physical product needs sticker sheets plus a placement board, backing substrate, printed grid/map, instructions, and packaging.

Use the terms `placement board`, `build board`, `backing board`, or `printed grid board`. Do not call this a LEGO baseplate.

## Placement Board / Backing Substrate

- Can you print or source a `12in` placement board?
- What board materials are available?
  - cardstock
  - poster board
  - foam board
  - rigid board
  - adhesive-safe printed board
- Can the board accept sticker adhesion without peeling or curling?
- Can the board ship flat without bending?
- Can it be rolled, folded, or tiled if needed?
- What is the board cost at sample, 10, and 100 units?

## Printed Grid / Map

- Can you print a numbered or color-coded `24x24` grid?
- Can section labels be printed clearly?
- Can coordinate labels be printed without clutter?
- Can you print black-base boards for First Hello?
- Can you print low-contrast guide lines that disappear visually after stickers are placed?

## Kit Packing

- Can you package board, sticker sheets, and instruction card together?
- Can sticker sheets be packed by build section?
- Can sticker sheets be packed by color?
- Can you include extra/spare stickers?
- Can you insert a printed build guide?
- Can you label each sheet or section?

## Future Sizes

- Can you support `16in` boards?
- Can you support `24in` designs as four `12in` tiled panels?
- Can panels align cleanly?
- What packaging changes are required?

## Magnet / Future Path

- Can you support magnetic backing later?
- Can you produce or source magnetic boards?
- What minimum thickness and shipping constraints apply?

## Decision

For first supplier outreach, sticker-sheet feasibility is primary.

Full-kit suppliers should also answer the placement-board and kit-assembly questions. No production order is being placed yet.
